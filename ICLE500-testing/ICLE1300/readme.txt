The contents of this folder are as follows:

- A pdf file explaining how the texts in the ICLE_1300 sample were selected and assessed
- Two spreadsheets (identical except that the xlsx file contains column name explanations) with the ICLE_1300 metadata and comparative judgement assessment scores
- An R project file, which should be opened prior to other R files
- An r folder containing:
-- A script created by Ian Jones for No More Marking, with functions supplementary to the sirt package in R. These are helpful for generating and analysing R models
-- A script for building the rating scale for the CJ assessments of the ICLE_1300 texts, plus the reliability of that scale
-- A report created by rendering that R script
- A data folder containing anonymised data generated during the CJ assessment task on No More Marking (www.nomoremarking.com)
